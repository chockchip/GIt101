////////////////////////////////////////////////////////////////////////////////
Initial Git
////////////////////////////////////////////////////////////////////////////////
1. Initial Repository

2. Setup username and email
git config --global user.name "rusucosmin" 
git config --global user.email "your-email@example.com"

3. Clone project
git clone https://github.com/dutylabs/git101

4. Create local git
git init

5. Clone project
git clone https://github.com/dutylabs/git101



////////////////////////////////////////////////////////////////////////////////
Check branch
////////////////////////////////////////////////////////////////////////////////
1. Check local branch
git branch 

2. Check remote branch
git branch -r

3. Check both remote and local branch
git branch -a



////////////////////////////////////////////////////////////////////////////////
Check status & log
////////////////////////////////////////////////////////////////////////////////
1. Check bracnh status
git status

2. Check log
git log

3. Generate log graph
git log --graph

4. Generate log graph (more pretty)
git log --graph --full-history --all --pretty=format:"%h%x09%d%x20%s"



////////////////////////////////////////////////////////////////////////////////
Add file and commit
////////////////////////////////////////////////////////////////////////////////
1. Add all change 
git add .



////////////////////////////////////////////////////////////////////////////////
Un-update before commit
////////////////////////////////////////////////////////////////////////////////
1. Un-add all file but file still keep modified
git reset

2. Un-add specific file
git reset "file name"

3. Un-add all file and revert it back to lastest track
git reset --hard



////////////////////////////////////////////////////////////////////////////////
Commit change
////////////////////////////////////////////////////////////////////////////////
1. Commit change
git commit -m "comment of commit"



////////////////////////////////////////////////////////////////////////////////
Put change from local to remote
////////////////////////////////////////////////////////////////////////////////
1. Push from current local to current remote
git push

2. Push specific branch from local to remote
git push origin "branch name"
cartoon_scrapping.py
3. Push specific local to speicific remote
git push origin daveleop:main


////////////////////////////////////////////////////////////////////////////////
Create remoter branch or track remote branch
////////////////////////////////////////////////////////////////////////////////
1. Create new remoter branch
git push --set-upstream origin "branch name" 




////////////////////////////////////////////////////////////////////////////////
Pull change from remote
////////////////////////////////////////////////////////////////////////////////
git pull origin "branchName"



////////////////////////////////////////////////////////////////////////////////
Change branch
////////////////////////////////////////////////////////////////////////////////
1. Change branch
git checkout "branchName"



////////////////////////////////////////////////////////////////////////////////
Delete branch
////////////////////////////////////////////////////////////////////////////////
1. Delete local branch
git branch -d "branchname"

2. Delete remote branch
git push origin -delete "branchname"



////////////////////////////////////////////////////////////////////////////////
Merge branch
////////////////////////////////////////////////////////////////////////////////
1. Merge specific branch to the current branch
git merge "branchName"



////////////////////////////////////////////////////////////////////////////////
Check different
////////////////////////////////////////////////////////////////////////////////
1. check different between branch
git diff origin/main main




////////////////////////////////////////////////////////////////////////////////
Stash
////////////////////////////////////////////////////////////////////////////////
Before stach have to add change first (git add .)

1. Save stash
git stash save "stashName"

2. Show stash list
git stash list

3. Use stash and remove from the list
git stash pop stash@{n}

4. Use stash but doesn't remove from the list
git stash apply stash@{n}



////////////////////////////////////////////////////////////////////////////////
Revert the commit
////////////////////////////////////////////////////////////////////////////////
- Revert the commit by refer to use hash number 
- Have to use git log for check hash number first 

1. Revert code
git revert "hashNumber"
